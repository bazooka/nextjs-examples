
A minimal template for NextJs projects.

Includes:
- Sass
- Extra image support (next-images)
- Eslint and stylelint with autofix support
- Script for generating components

### Develop

Install dependencies:

```
npm install
```

---

Run dev environment:

```
npm run dev
```

---

Start component generator script:

```
npm run make
```

Create or change component templates in `tools/make/templates`.
